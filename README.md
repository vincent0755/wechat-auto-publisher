# 公众号自动发布助手

这是一个 Windows 桌面软件成品雏形。用户填入自己的公众号 API 信息，选择封面图，添加正文图片，写普通文字，软件会自动排版并调用公众号官方接口创建草稿，也可以选择立即提交发布。

## 已实现功能

- 填写 `AppID` / `AppSecret`
- 可选记住 `AppSecret`，仅保存到本机配置文件
- 一键快速检测当前公网 IP，多服务并发查询，减少等待
- 测试连接失败时解析“微信接口看到的 IP”，方便加入白名单
- 一键复制公网 IP，方便填入公众号后台白名单
- 记录上次公网 IP，发布前自动检查是否变化
- 测试公众号 API 连接，并显示成功/失败弹窗、错误原因和测试时间
- 选择文章封面图
- 添加多张正文图片，并自动穿插到正文段落中
- 内置文章模板：日常分享、活动通知、产品种草、知识干货
- 本地保存和加载草稿
- 在正文中使用 `[[图片1]]`、`[[图片2]]` 插入图片，并支持 `[图片1]`、`[[图1]]`、`第1张图`、`图片1放这里` 等写法
- 自动把普通文字转换为公众号 HTML 排版
- 可选添加少量装饰表情，默认公众号图文版式会尽量克制
- 支持三种表情风格：温暖、活泼、商务
- 支持三种排版主题：清新蓝、暖橙、商务灰
- 支持四种版式：公众号图文、杂志风、轻卡片、简约线条
- 默认生成干净公众号正文，仅对明确的小标题、引用、列表和图片占位符做排版
- 富文本工具栏：支持大标题、小标题、加粗、斜体、引用、列表、图片占位符和清除格式
- 本地生成排版预览
- 内置 AI 写作：填写 OpenAI API Key、主题、读者和语气后自动生成标题、摘要、正文
- 剪贴板导入：从 ChatGPT 网页复制内容后，一键导入到软件正文编辑区
- 发布准备状态条：实时提示账号、标题、封面、正文、图片和 IP 是否准备好
- 正文、标题、摘要、AI 要求和操作反馈支持右键复制、粘贴、全选
- 发布前检查清单：账号、标题、封面、正文、图片占位符、原文链接
- 图片检查：格式、文件大小、尺寸、封面比例
- 批量上传封面图和正文图片
- 创建公众号草稿
- 可选：创建草稿后立即提交发布；不勾选时只会进入公众号后台草稿箱
- 发布完成后显示并可复制 `media_id` / `publish_id`，方便排查
- 保存常用项：`AppID`、作者、上次标题
- 默认不保存 `AppSecret`；勾选“记住 AppSecret”后才保存，可随时清除

## 正文排版语法

直接写普通文字即可。软件支持这些简单标记：

```text
## 二级标题
### 三级标题
> 引用文字
- 列表项
--- 分隔线
[[图片1]]
```

发送前会自动生成适合公众号文章的 HTML。

编辑区上方有“富文本”工具栏，可以选中文字后点击：

- 大标题 / 小标题
- 加粗 / 斜体
- 引用 / 列表
- 插入图片位
- 清除格式

编辑区会即时显示标题、引用、列表、图片占位符等样式；发布时仍会转换成公众号可用的 HTML。

图片位置最准确的写法是把占位符单独放一行：

```text
这里是上一段正文。

[[图片1]]

这里是下一段正文。
```

也支持这些写法，软件会在预览和发布前自动转换：

```text
[图片1]
[[图1]]
第1张图
图片1放这里
这里插入第一张图
```

## 装饰表情

软件默认不依赖表情和符号来撑版面。“公众号图文”版式会尽量保持干净，即使勾选装饰表情，也不会在正文里乱加符号。

如果选择其他更活泼的版式，可以勾选装饰表情，用于小标题、引用或列表的轻微点缀。

可选风格：

- 温暖：适合日常分享、生活号、社群号
- 活泼：适合营销活动、种草内容、年轻用户
- 商务：适合企业通知、项目总结、知识内容

如果不想加表情，可以取消勾选“装饰表情”。

## 干净排版

默认“公众号图文”版式以干净、稳定为主：

- `## 二级标题` 会变成简洁左侧色条标题
- `### 三级标题` 会变成加粗小标题
- `> 引用文字` 会变成简洁引用段
- `- 列表项` 会变成普通项目列表
- `[[图片1]]` 会按指定位置插入图片
- 普通段落不会因为出现“重点、时间、小结”等词自动变成文字框
- 软件不再根据关键词猜测加框，避免把正常正文排乱

可选排版主题：

- 清新蓝
- 暖橙
- 商务灰

可选版式：

- 公众号图文：默认推荐，更接近人工编辑正文，不强加栏目分类、图片编号或列表编号
- 杂志风：栏目感更强，适合专题文章
- 轻卡片：浅色卡片更多，适合活动、产品种草
- 简约线条：留白更多，适合知识干货和企业内容

## 模板和草稿

编辑区顶部可以选择文章模板：

- 日常分享
- 活动通知
- 产品种草
- 知识干货

点击“套用模板”会把正文替换为对应结构，用户只需要把占位内容改成自己的内容。

点击“添加正文图片”后，可以一次选择多张图片。默认开启“添加多张图片后自动穿插到正文段落”。

插图优先级：

1. 如果正文里已经写了 `[[图片1]]`、`[[图片2]]`，软件会完全尊重这些位置。
2. 如果正文里写了 `[图片1]`、`[[图1]]`、`第1张图`，软件会先转换成标准占位符。
3. 如果正文里写了“这里放图”“插入图片”“配图”“放二维码”“放海报”等提示，软件会把提示行替换成图片占位符。
4. 如果正文里写了“这里放第2张图”“插入图片2”“图片1放这里”“这里插入第一张图”，软件会按指定编号插入对应图片。
5. 如果没有任何插图要求，软件才会按段落长度自动分布图片。

发布时，软件会批量上传正文图片，并把这些占位符替换成微信返回的图片地址。如果用户手动指定了部分图片位置，软件不会擅自移动；缺少的图片占位符会在日志里提醒。

也可以点击“自动插图”，让软件重新按正文段落分布所有正文图片。

点击“保存草稿”会保存当前文章信息、正文、封面路径、正文图片路径和排版设置。点击“加载草稿”可以继续编辑。

草稿默认保存在本机用户数据目录：

```text
%APPDATA%\WechatAutoPublisher\drafts
```

## AI 写作和剪贴板导入

左侧切换到“AI写作”标签页，可以使用两种方式把 GPT 内容放进软件：

1. 内置 AI 写作  
   填写 OpenAI API Key、文章主题、目标读者、语气和补充要求，然后点击“AI生成文章”。软件会自动把生成结果拆成标题、摘要和正文，并放入编辑区。

2. 剪贴板导入  
   在 ChatGPT 网页或其他工具里复制文章内容，然后回到软件点击“从剪贴板导入”。如果内容里包含“标题：”“摘要：”“正文：”，软件会自动识别；如果没有，也会把正文放入编辑区。

OpenAI API Key 默认不会保存。只有勾选“记住 OpenAI Key（仅保存在本机）”时，才会写入本机配置文件。

## 使用前准备

1. 在微信公众平台启用开发者能力，拿到 `AppID` 和 `AppSecret`。
2. 在软件里点击“检测 IP”，复制显示出来的公网 IP。
3. 在公众号后台配置 API 调用 IP 白名单。
4. 如果只需要草稿，不要勾选“创建草稿后立即提交发布”；草稿创建成功后到公众号后台“内容与互动 -> 草稿箱”查看。
5. 如果要“立即发布”，请勾选“创建草稿后立即提交发布”，并确认公众号账号具备发布接口权限。

公众号后台路径通常是：

```text
微信公众平台 -> 设置与开发 -> 基本配置 -> IP 白名单
```

白名单要填的是运行软件这台电脑调用微信接口时的出口公网 IP。软件里的“检测 IP”是第三方查询结果，通常是对的，但在代理、VPN、公司网络、双线路出口、运营商 NAT 场景下，第三方看到的 IP 可能和微信接口看到的 IP 不一致。

如果点“测试连接”提示 `40164` 或“不在白名单”，请优先看软件里显示的“微信看到 IP”，复制这个 IP 加入公众号后台白名单。

如果 IP 变化了，请先把新的 IP 加到公众号后台白名单，再发布。

注意：如果软件直接装在朋友电脑上运行，微信接口请求会从朋友当前公网 IP 发出；公众号后台的 IP 白名单也要配置这个 IP。若要稳定给很多朋友使用，建议升级为“云端固定 IP 服务 + 桌面客户端”的架构。

## 本地运行

最方便的方式：双击项目目录里的：

```text
打开软件.bat
```

也可以用命令启动：

```powershell
python app.py
```

软件底部提供三个快捷入口：

- 打开草稿目录
- 打开软件目录
- 使用说明

软件顶部提供“发布准备”状态条，可以直接看到当前还缺哪些内容。点击“检查准备状态”会执行完整发布前检查，包括图片格式、大小、尺寸和占位符匹配。

## 打包为 exe / 安装包

```powershell
.\build_windows.ps1
```

脚本会先用 PyInstaller 生成：

```text
dist\WechatAutoPublisher.exe
```

如果电脑已安装 Inno Setup 6，还会生成：

```text
output\WechatAutoPublisherSetup.exe
```

这个安装包可以分享给朋友使用。
